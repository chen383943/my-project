As considering the value of the used car, we found an obvious problem with the year relates to the price. In reality, when we are considering buying a car, we will consider the car_age instead of only the year that it made. To make sure this factor is considered, we make a new column call car_age to represent the car age ehrn the car sale.

In [104]:
case['posting_date'] = pd.to_datetime(case['posting_date'], format='%Y-%m-%dT%H:%M:%S%z', utc=True)
case2['posting_date'] = pd.to_datetime(case2['posting_date'], format='%Y-%m-%dT%H:%M:%S%z', utc=True)

case['posting_year'] = case['posting_date'].dt.year
case2['posting_year'] = case2['posting_date'].dt.year

case['car_age'] = case['posting_year'] - case['year']
case2['car_age'] = case2['posting_year'] - case2['year']
case=case.reset_index(drop=True)
case2=case2.reset_index(drop=True)
